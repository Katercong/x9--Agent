# 文件地图

> 代码入口 + 各文件一句话描述。新会话不要 grep 整个项目，从这份开始。

## 整体目录

```
F:\Claude_Project\Database\
├─ database.db              主库（SQLite）— 单一可信源
├─ schema.sql               初始建表（v1.0）— 之后用 migrate_v* 增量
├─ run.bat / stop.bat       启停脚本（Windows）
├─ restart.bat              一键重启
├─ reset_key.bat            紧急重置 user key 的 CLI
├─ reimport.bat             重跑数据导入
├─ .api_key                 张原始 token（备份用，可删）
├─ .local_keys_backup.txt   张+廖初始 token（gitignore 内）
├─ .gitignore               敏感文件白名单
├─ app/                     后端 + 前端
├─ scripts/                 一次性脚本 + 调度模板
├─ assets/                  图片资源
├─ exports/                 自动生成的 json/xlsx 镜像
└─ docs/                    所有项目文档
```

## `app/` — 后端 (FastAPI 模块)

| 文件 | 职责 |
|---|---|
| `main.py` | FastAPI 入口；挂载所有 router；遗留 `/api/products` 等老端点；`/docs/{name}` 多格式查看器 |
| `auth.py` | 鉴权核心：SHA-256 哈希 + `require_authenticated` / `require_user_or_above` / `require_admin` 三档依赖 |
| `auth_routes.py` | `/api/v1/auth/*` 端点：whoami / 用户管理 / Key 签发与撤销 |
| `registry.py` | 资源注册（`_meta_resource`）+ 内置命名查询定义（13 条）+ Resource 数据类 |
| `v1.py` | 通用 `/api/v1/*` 路由：data CRUD、tables 建表、queries 命名查询 |
| `llm.py` | LLM 调用统一代理：Anthropic + OpenAI-compat 适配；feature binding (`get_provider_for_feature`) |
| `agent.py` | AI 项目助手（运维咨询）— 加载所有 docs 作 system prompt + 动作链接路由 |
| `outreach_ai.py` | 邀约话术生成器（任务 3.2.1） |
| `title_optimizer.py` | TK 标题热搜优化器（任务 2.2.2） |
| `keyword_ai.py` | 热搜分析层：dashboard + trend + enrich（自动分类） |
| `static/index.html` | 前端单页：所有 tab + 悬浮 AI 助手 + 登录浮层 |

## 端点速查（高频）

```
GET  /api/v1/                          discovery
GET  /api/v1/auth/whoami               当前登录人
POST /api/v1/data/{r}/bulk             通用批量 upsert
PATCH /api/v1/data/{r}/{id}            通用更新
GET  /api/v1/queries/{name}            跑命名查询
POST /api/v1/queries                   加新查询（admin 才能加）
POST /api/v1/tables                    自助建表（admin 才能建）
POST /api/v1/llm/complete              统一 LLM 调用（廖用）
POST /api/v1/ai/optimize_title         标题优化
POST /api/v1/ai/generate_outreach      邀约话术生成
POST /api/v1/ai/keywords/enrich        关键词 AI 分类
GET  /api/v1/keywords/dashboard        热搜实时仪表盘
POST /api/v1/agent/chat                AI 助手
GET  /docs/{name}                      多格式文档查看器
```
