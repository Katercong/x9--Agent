## 业务

| 词 | 含义 |
|---|---|
| **X9** | 我们这个跨境品牌名。Clean, Light & Free。覆盖女性护理 / 母婴 / 成人护理 / 宠物护理 |
| **SKU** | 产品编码。如 `BU02P155`（女性护垫）、`EU06FDXS`（宠物纸尿裤） |
| **达人 / KOC** | TikTok 创作者。KOC = Key Opinion Consumer，比 KOL 更小更草根 |
| **creator (单数表)** | X9 BD **主表**, INTEGER id, 张维护已确认达人 + outreach 流水的 FK 来源（66 行）|
| **creators (复数表) / creator_leads** | 廖爬虫 **lead 池**, VARCHAR(120) id, 含 priority/fit/scoring 字段。**URL slug 是 `creator_leads`**（避让主表）。等 ETL 后会 promote 进 `creator` 主表 |
| **中腰部 KOC** | 1K-50W (1000-500000) 粉丝的达人，按需求确认表 3.1.3 |
| **建联** | BD 跟达人发私信 / 邮件 邀约合作 |
| **寄样** | 给达人寄产品样品测评 |
| **挂车** | 达人在 TT 视频里挂产品链接 |
| **Spark Ads** | TikTok 给品牌买达人视频做广告投放的功能 |
| **Spark Ads code / 授权码** | 达人给品牌的视频投放授权码 |
| **GMV** | Gross Merchandise Volume，总交易额 |
| **PPS** | 内容表现分（Performance Per Stream）从 TT Creator Marketplace |
| **engagement_rate / 互动率** | (likes + comments + shares) / views，0~1 |
| **挂车 vs 引流独立站** | 挂车 = TT 内购买；引流 = 跳到 x9x9.us 站外买 |

## 技术

| 词 | 含义 |
|---|---|
| **upsert** | INSERT 不存在就插，存在按 unique key 更新 |
| **bulk** | 批量端点，一次接受 array of items |
| **resource** | 注册到 `_meta_resource` 表的资源，有通用 CRUD |
| **named query** | 注册到 `_meta_query` 表的预定义 SQL，给前台/廖一键调 |
| **feature binding** | `llm_feature` 表里某个 AI 功能绑定具体 Provider+Model |
| **fallback** | 不绑模型时走全局 active provider |
| **few-shot** | 喂给 LLM 的"示例样本"（这里用历史话术做） |
| **outbox** | 半自动触达队列：AI 生成 → BD 复制 → 标已发 |
| **bootstrap seed** | 项目初始的占位数据（24 条 TK 热搜词都是），等真实数据覆盖 |
| **trigger** | SQLite 触发器。本项目 4 个：snapshot×2 / auto_category / updated_at×2 |
| **migrate_v***  | 一次性升级脚本，幂等。最新到 v10 |

## 文件命名约定

| 前缀 | 含义 |
|---|---|
| `migrate_v*.py` | 数据库 schema 升级 |
| `import_*.py` | 一次性数据导入（产品 / 达人 / 图片 / 话术历史） |
| `export_*.py` | 镜像导出（json / xlsx） |
| `app/*.py` | FastAPI 模块，每个功能一个 |
| `docs/*.md` | 项目文档，agent 加载 |
| `scripts/*.py` | 一次性 / 周期性脚本（不是 server） |
| `*.bat` | Windows 启动器 |

## 配置位置

| 配置 | 在哪 |
|---|---|
| 用户和 Key | `api_user` + `api_key` 表 |
| LLM Provider | `llm_provider` 表 |
| AI 功能绑模型 | `llm_feature` 表 |
| 业务参数（佣金 / 寄样 / 禁词） | `app_config` 表 |
| 命名查询 SQL | `_meta_query` 表 |
| 资源注册（通用 CRUD） | `_meta_resource` 表 |
| 张和廖的初始 token 备份 | `.local_keys_backup.txt` |
