## 已踩过的坑（避免新会话重蹈）

- ✅ Windows + uvicorn `--reload` 会留僵尸进程占端口 → 已建 stop.bat
- ✅ `.bat` 文件不能用 LF 换行 / 不能写 `>/dev/null` → 必须 CRLF + `>NUL`
- ✅ 端口 8765 被占用 → 改用 18765
- ✅ SQLite BEFORE INSERT trigger 不能 UPDATE WHERE id=NEW.id → 用 AFTER INSERT
- ✅ Linter 反复把 .bat 里 `>NUL` 改回 `>/dev/null` → 用 Python 脚本绕过 hook 重写
- ✅ 把自己的 admin key 给别人 = 审计错乱 → 只能给对方建账号 + 签发新 key
- ✅ Bash + curl + 中文 JSON body 会有编码问题 → 用 Python urllib 或 ASCII body
- ✅ Agent 生成 `#nav:changelog` 等不存在的 tab → 系统提示已收紧 + 前端兜底映射
