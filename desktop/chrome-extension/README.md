# TikTok Creator Lead Browser v2.2

X9 采集插件统一连接线上服务：

```text
https://usx9.us
```

运行时只使用线上服务。

Shop 上传接口：

```text
https://usx9.us/api/local/collector/observations
```

使用步骤：

1. 打开 `chrome://extensions`，开启开发者模式。
2. 重新加载本目录插件。
3. 打开 `https://affiliate-us.tiktok.com/connection/creator`。
4. 在插件里点击 TikTok Shop 全自动采集的开始按钮。
5. 到 `https://usx9.us/portal/collection` 查看采集结果。
