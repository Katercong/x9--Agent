## 发送前检查清单

发邮件之前,请你确认:

- [ ] 密钥已经吊销 + 重置(见 00 文档 🚨 第 1 件)
- [ ] GitHub 仓库已经配好分支保护(见 02 文档)
- [ ] `.env.shared.example` 已经提交到仓库
- [ ] CI workflow 已经能跑通(至少绿一次)
- [ ] CODEOWNERS 里的 `@your-github-username` 都替换成真实账号
- [ ] 准备好至少 1-2 个 `good-first-issue` 标签的 issue,给新人练手
- [ ] 把这份邮件里的 `<...>` 占位符都填好
- [ ] 把新的 `.env.shared`(含新密钥)通过安全渠道(微信私聊 / 1Password / Bitwarden)发给每个人,**不要走邮件**
