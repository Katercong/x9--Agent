# 运营实习生插件测试-极简步骤

## 1. 谁启动服务

服务只在「服务器/同事电脑」启动，实习生电脑不部署服务。

在服务器/同事电脑打开 PowerShell：

```powershell
cd F:\Claude_Project\X9_AI_system_fresh
git fetch origin
git checkout feat/foreign-trade-system
git pull
.\start_all.ps1
```

启动后确认能打开登录页：

- 本机测试：`http://localhost:8000/login`
- 实习生电脑测试：`http://服务器IP:8000/login` 或正式测试域名

## 2. 实习生电脑准备

只需要 Chrome 浏览器。

不要安装 Python、Node、Docker，也不要运行 `start_all.ps1`。

准备两个账号：

- 跨境部账号：测试原 TK 插件
- 外贸部账号：测试外贸插件

## 3. 测试跨境部 TK 插件

1. 实习生电脑打开系统登录页。
2. 用「跨境部账号」登录。
3. 点击左侧「下载插件」。
4. 解压下载的 `x9-tk-creator-extension.zip`。
5. Chrome 打开 `chrome://extensions/`。
6. 打开「开发者模式」。
7. 点「加载已解压的扩展程序」，选择刚解压的插件文件夹。
8. 打开 TikTok 或 TikTok Shop 采集页面。
9. 打开插件侧边栏，执行一次小范围采集。
10. 回系统看「采集总览 / TikTok Shop / X9 线索」是否出现新记录。

通过标准：

- 插件能打开。
- 不要求先绑定账号也能开始采集。
- 系统里能看到新采集记录或数量增加。

## 4. 测试外贸部插件

先在 `chrome://extensions/` 禁用或移除刚才的 TK 插件，避免两个插件同时干扰。

1. 实习生电脑打开系统登录页。
2. 用「外贸部账号」登录。
3. 点击左侧「下载插件」。
4. 解压下载的 `x9-foreign-trade-extension.zip`。
5. Chrome 打开 `chrome://extensions/`。
6. 点「加载已解压的扩展程序」，选择外贸插件文件夹。
7. 打开招聘网站页面，执行一次招聘线索采集。
8. 回系统看「采集 · 招聘网站」或「公司客户线索」是否出现新记录。
9. 打开小红书或抖音页面，执行一次社媒线索采集。
10. 回系统看「采集 · 小红书抖音」或「社媒线索」是否出现新记录。

通过标准：

- 外贸账号下载到的是外贸插件。
- 招聘采集能上传。
- 小红书/抖音采集能上传。
- 外贸部页面能看到新记录或数量增加。
